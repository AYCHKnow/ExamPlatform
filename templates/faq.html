This Project

As you may have noticed, this isn't a fully working application. This is a final project for the Front-End Development part of the Full-Stack Software Developer course from Code Institute. This is a showcase of the skills I've learned so far and kind of a proof of concept

  • live_helpWhat is this app for?

    It's for all your multiple choice quizzing and examination needs :) This is a front-end web development project for a hypothetical examination app.

  • live_helpWhat does it do?

    It contains informational views about the application and services offered, such as: faq, pricing, exam list, & login/registration forms. It also allows you to take a sample interactive exam and provide be provided with score report at end

    The idea is that the complete app would allows organizations to host their multiple choice exams/quizzes and users could register to take them. However, no backend for this to be possible is going to be implemented at this stage. This is more of a proof of concept with focus on front end and set of of AngualrJS templates, controllers, routes, and services.

  • live_helpHow does it work?

    This is a Single Page Application on AngularJS and uses JSON data to generate exam questions and answers. The site is styled with MaterializeCSS.

  • live_helpWhy AngularJS?

    Some Amazing AngularJS Features:

    • Improved User Experience
    • HTTP Communication with XHR (XML Http Request)
    • DOM Manipulation
    • State and Routing
    • MVC (MVVM in this case)

    Benefits:

    • Two-Way Data Binding
    • Many ways to do things
    • Good Documentation
    • Large Community
    • Can get very productive one you get familiar

    Some Downsides though:

    • Steep Learning Curve (At least I thinks so :)
    • Lots of unfamiliar terms)
    • Many ways to do things
    • SEO ?

    Overall, benefits outweigh the downsides!
